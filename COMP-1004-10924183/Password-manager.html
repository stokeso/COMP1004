Passwords

recommendations when making a password:

Password length: most passwords should contain a minimum of 8 characters to add to complexity however a strong password should have 15 characters but this could lead to a harder to recall password.

Password composition: a good password should include at least one upper and lowercase, numeric, and special character (#, !, ?, $).

Password expiration: passwords do not need to be chnaged on a regular interval however a change should be enforced if a data breach occurs.
